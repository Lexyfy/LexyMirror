Projects

Featured project

IHK Project: Secure User System

My main showcase project is a user system developed during my IHK final project phase, with a strong focus on security, structure, and practical usability.

The core of the project was a secure authentication flow with password hashing, two-factor authentication via email and app, JWT-based sessions, and a clearly modular backend structure.

The goal was not just to create a working login system, but to build an extendable application that is technically understandable, clearly documented, and suitable for practical use.

Technical focus

  • Back-End: Node.js, Express, and a modular route and middleware structure
  • Security: password hashing, two-factor authentication, and validation
  • Session logic: JWT-based authentication
  • Database: MongoDB for managing user data and application logic

Why it matters in my portfolio

This project is currently the strongest example of how I combine technical requirements, security aspects, and clean structure in a real development context.

It deliberately replaces older mini-projects as the main focus of my portfolio, because it represents my current level far more accurately.
